#Site web de Patrick Bacon

Ce projet héberge le code source du site web de Patrick Bacon.





## Premier apercus du site 

Le site est une application web d'une seule page. 
Le projet est divisé en 3 partie:

* Un backend accesible uniquement en REST.

* Un frontend optimisé pour la rapidité, multiplateforme, respectant les meilleurs pratiques du web et mettant de l'avant le AJAX.

* Un côté admin efficace pour la gestion du contenu et conviviale.



Voici, en résumé, avec quoi le projet est construit:

* Pour le backend: Symfony 2.8

* Pour le frontend et le côté admin:

	* Twig *(Symfony template engine)*,
	
	* LESS,
	
	* Bootstrap 3.3.7, 
	
	* JQuery 3.2, 
	
	* Fontello *(pour les icones)*
	
	* Grunt,
